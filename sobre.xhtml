Wizard Games

Empresa fictícia que foi criada para elaboração do trabalho da disciplina "Marcação e Layout para Web" ministrada pelo docente Luiz Gustavo Maschietto, esta integrante do curso de Tecnologia em Sistemas para Internet oferecido pelo Instituto Federal de Educação, Ciência e Tecnologia São Paulo - Campus São João da Boa Vista (site).


Nossa sede fica na Suíça, Suíça, Liechtenstein, Vaduz - 9490


O projeto consiste no uso de HTML, CSS, JavaScript etc, para elaboração de um site com o tema desejado, seguindo uma lista de itens mínimos.
